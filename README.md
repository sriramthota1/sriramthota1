## 🚀 Featured Projects

### 🧠 Brain Tumor Detection System
Deep learning-based computer vision system for detecting and classifying brain tumors from medical imaging data.

- Built an object detection pipeline using YOLO-based architecture for tumor localization and classification  
- Applied preprocessing, augmentation, and model evaluation workflows for medical image datasets  
- Evaluated model performance using accuracy, precision, recall, F1-score, and confusion matrices  

### 🍅 Tomato Leaf Disease Detection & Classification
Computer vision system for detecting tomato leaf diseases using deep learning models.

- Trained and compared CNN-based architectures including YOLOv8, MobileNetV2, and ResNet50  
- Used image augmentation, preprocessing, and evaluation pipelines to improve generalization  
- Designed experiments for model comparison, accuracy tracking, and inference performance  

### 🧬 AI-based E. coli Detection using UV-C Fluorescence
Research project focused on using computer vision and deep learning for bacterial detection.

- Built ML pipelines for fluorescence imaging data preprocessing, model training, and evaluation  
- Applied deep learning models for detection and concentration estimation across experimental surfaces  
- Evaluated performance using accuracy, precision, recall, F1-score, and inference metrics  

### 🧑‍💻 TrackMyAttendance
Real-time facial recognition attendance system using computer vision and full-stack development.

- Developed an AI-powered attendance platform using Python, OpenCV, Flask, and MongoDB  
- Implemented face detection, recognition, attendance logging, and admin dashboard workflows  
- Designed the system for real-time usage with automated attendance tracking  

### 🤖 OpsSync — Agentic AI Workflow Platform
AI workflow automation platform using LLMs, retrieval, APIs, and backend orchestration.

- Built agentic workflows using LangChain, FastAPI, vector search, and tool-based execution  
- Integrated RAG pipelines, memory, prompt orchestration, and response validation  
- Containerized services using Docker and deployed scalable workflow components
